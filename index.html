HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading tags h1 - h6 should be used in order.
  • Semantic html tags should be used whenever possible for better code organization as well as accessibility.
  • The img tag is a self closing tag.
  • html is organized into a family tree structure.
  • html attributes consist of a name and value. (name="value")
